新型快递包装箱的整体设计结构为拼接结构，一个新型快递包装箱一般由6个拼接板拼接而成。各个拼接板的大小由其是由几个拼接单元组成决定。每个拼接单元都由主板和副翼。其中，主板正面设有凸扣和保护层，背面也设有凸扣；副翼是可摆动的，正反面也都设有凹扣。

绿色快递包装袋

新型快递包装袋在普通购物袋的基础上，将塑料袋主体的一侧延伸至提手顶端处，以便于在另一侧用快递单进行封口。在提手处用易撕线封口，用于保护物品的私密性，在进行二次利用时可沿此线撕开作为提手，简单便捷。为实现循环利用，袋子主体使用强度韧性较高的材质pe聚乙烯，主体延伸以及封口处周围使用耐磨塑料PVC，保证多次粘取不易损坏。
